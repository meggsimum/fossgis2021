Massenkonvertieren kartographischer Stildokumente mit GeoStyler-CI


Jan Suleiman, Daniel Koch, terrestris GmbH & Co. KG Christian Mayer, meggsimum - Büro für Geoinformatik

FOSSGIS 2021, 08.06.2021

GeoStyler

  • Gebrauchsfertige JS Bibliothek zum Styling für Geodaten
  • Unterstützte Formate
    • SLD, Mapfiles, Mapbox, QGIS, OpenLayers ...
  • Umfangreiche webbasierte UI Komponenten
  • geostyler.org

GeoStyler CLI

  • Überführung von Stildokumenten in andere Formate
  • Voraussetzung: NodeJS
  • Installation

              $ npm install -g geostyler-cli

GeoStyler CLI

Konvertierung Ordner mit Stildokumenten

geostyler [options] /path/to/input/ 

              # converts all Mapfiles in the given folder to SLD
              $ geostyler -s mapfile -t sld -o /path/to/output /path/to/input/
